## مجموعه T3

مجموعه T3 یک مجموعه از ابزارها و کتابخانه‌ها است که برای ساخت برنامه‌های وب مدرن با استفاده از Next.js و TypeScript طراحی شده است. این مجموعه شامل:

- [Next.js](https://nextjs.org/) - یک فریم‌ورک React برای ساخت برنامه‌های وب
- [TypeScript](https://typescriptlang.org/) - یک زبان برنامه‌نویسی که JavaScript را با ایمنی تایپ گسترش می‌دهد
- [Tailwind CSS](https://tailwindcss.com/) - یک فریم‌ورک CSS برای ساخت رابط‌های کاربری زیبا
- [tRPC](https://trpc.io/) - یک کتابخانه API برای ساخت API‌های ایمن از نظر تایپ
- [Prisma](https://prisma.io/) - یک ORM برای تعامل با پایگاه داده
- [NextAuth.js](https://next-auth.js.org/) - یک کتابخانه احراز هویت برای Next.js

اگر می‌خواهید در مورد هر یک از این ابزارها بیشتر بدانید، لطفاً مستندات رسمی آنها را بررسی کنید.

import Callout from "../../components/docs/callout.tsx";
import CompanyList from "../../components/docs/companyList.tsx";
import OpenSourceAppList from "../../components/docs/openSourceAppList.tsx";
